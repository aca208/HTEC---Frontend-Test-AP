Step 1:
Download and install Node.js
https://nodejs.org/dist/v8.11.1/node-v8.11.1-x64.msi

Step 2:
After succesful installation, open cmd(Command Prompt Commands) and run a command "npm -v".
It should return a version of npm installed if it was a succesful installation.

Step 3:
Run a command in cmd "npm install http-server -g" and wait till it finishes.

Step 4:
After the installation of http-server run a command "http-server [path]", where path is the absolute path of root folder of the web app.
